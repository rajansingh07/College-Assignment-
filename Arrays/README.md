# Arrays

This folder contains Java solutions for common array-based DSA problems.

## Problems

| File | Problem | Approach | Time | Space |
|------|---------|----------|------|-------|
| [TwoSum.java](TwoSum.java) | Two Sum | HashMap | O(n) | O(n) |
| [MaxSubarraySum.java](MaxSubarraySum.java) | Maximum Subarray (Kadane's) | Sliding window | O(n) | O(1) |

## How to Run

```bash
cd Arrays
javac TwoSum.java && java TwoSum
javac MaxSubarraySum.java && java MaxSubarraySum
```
