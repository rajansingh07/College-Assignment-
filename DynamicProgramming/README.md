# Dynamic Programming

This folder contains Java solutions for common dynamic programming (DP) problems.

## Problems

| File | Problem | Approach | Time | Space |
|------|---------|----------|------|-------|
| [Fibonacci.java](Fibonacci.java) | Fibonacci Number | Memoization / Tabulation / Space-optimized | O(n) | O(1) |
| [LongestCommonSubsequence.java](LongestCommonSubsequence.java) | Longest Common Subsequence | 2-D DP table | O(m·n) | O(m·n) |

## How to Run

```bash
cd DynamicProgramming
javac Fibonacci.java && java Fibonacci
javac LongestCommonSubsequence.java && java LongestCommonSubsequence
```
