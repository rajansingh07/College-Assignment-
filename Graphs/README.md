# Graphs

This folder contains Java solutions for common graph-based DSA problems.

## Problems

| File | Problem | Approach | Time | Space |
|------|---------|----------|------|-------|
| [BFS.java](BFS.java) | Breadth-First Search | Queue + visited array | O(V+E) | O(V) |
| [DFS.java](DFS.java) | Depth-First Search | Recursive + visited array | O(V+E) | O(V) |

## How to Run

```bash
cd Graphs
javac BFS.java && java BFS
javac DFS.java && java DFS
```
