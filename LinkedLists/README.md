# Linked Lists

This folder contains Java solutions for common linked list DSA problems.

## Problems

| File | Problem | Approach | Time | Space |
|------|---------|----------|------|-------|
| [ReverseLinkedList.java](ReverseLinkedList.java) | Reverse Linked List | Iterative (3 pointers) | O(n) | O(1) |
| [DetectCycle.java](DetectCycle.java) | Detect Cycle | Floyd's Tortoise & Hare | O(n) | O(1) |

## How to Run

```bash
cd LinkedLists
javac ReverseLinkedList.java && java ReverseLinkedList
javac DetectCycle.java && java DetectCycle
```
