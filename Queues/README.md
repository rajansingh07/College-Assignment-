# Queues

This folder contains Java solutions for common queue-based DSA problems.

## Problems

| File | Problem | Approach | Time | Space |
|------|---------|----------|------|-------|
| [CircularQueue.java](CircularQueue.java) | Circular Queue (Ring Buffer) | Array + two pointers | O(1) | O(k) |
| [QueueUsingStacks.java](QueueUsingStacks.java) | Queue using Two Stacks | Lazy transfer | Amortized O(1) | O(n) |

## How to Run

```bash
cd Queues
javac CircularQueue.java && java CircularQueue
javac QueueUsingStacks.java && java QueueUsingStacks
```
