# DSA Practice — Java Solutions

This repository contains Data Structures and Algorithms (DSA) practice solutions in Java, organized by topic for easy reference and revision. Problems are sourced from LeetCode, CodeChef, and standard textbooks.

## Repository Structure

```
.
├── Arrays/                  # Array manipulation problems
├── Strings/                 # String processing problems
├── LinkedLists/             # Singly linked list problems
├── Stacks/                  # Stack-based problems
├── Queues/                  # Queue-based problems
├── Trees/                   # Binary tree problems
├── Graphs/                  # Graph traversal problems
└── DynamicProgramming/      # DP problems
```

## How to Run Any Solution

Each Java file is self-contained with a `main` method that prints test results.

```bash
# Example — compile and run from the repo root:
cd Arrays
javac TwoSum.java && java TwoSum
```

Requires **Java 8+**. No external dependencies.
