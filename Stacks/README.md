# Stacks

This folder contains Java solutions for common stack-based DSA problems.

## Problems

| File | Problem | Approach | Time | Space |
|------|---------|----------|------|-------|
| [ValidParentheses.java](ValidParentheses.java) | Valid Parentheses | Stack | O(n) | O(n) |
| [MinStack.java](MinStack.java) | Min Stack | Two stacks | O(1) | O(n) |

## How to Run

```bash
cd Stacks
javac ValidParentheses.java && java ValidParentheses
javac MinStack.java && java MinStack
```
