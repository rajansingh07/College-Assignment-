# Strings

This folder contains Java solutions for common string-based DSA problems.

## Problems

| File | Problem | Approach | Time | Space |
|------|---------|----------|------|-------|
| [ReverseString.java](ReverseString.java) | Reverse String | Two pointers | O(n) | O(1) |
| [ValidAnagram.java](ValidAnagram.java) | Valid Anagram | Frequency count | O(n) | O(1) |

## How to Run

```bash
cd Strings
javac ReverseString.java && java ReverseString
javac ValidAnagram.java && java ValidAnagram
```
