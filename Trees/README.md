# Trees

This folder contains Java solutions for common tree-based DSA problems.

## Problems

| File | Problem | Approach | Time | Space |
|------|---------|----------|------|-------|
| [BinaryTreeTraversals.java](BinaryTreeTraversals.java) | Binary Tree Traversals (Inorder/Preorder/Postorder/BFS) | Recursive DFS + BFS | O(n) | O(n) |
| [MaxDepth.java](MaxDepth.java) | Maximum Depth of Binary Tree | Recursive DFS | O(n) | O(h) |

## How to Run

```bash
cd Trees
javac BinaryTreeTraversals.java && java BinaryTreeTraversals
javac MaxDepth.java && java MaxDepth
```
